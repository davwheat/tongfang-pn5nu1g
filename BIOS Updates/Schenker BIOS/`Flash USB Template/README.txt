Instructions
============

**ALL SCHENKER BIOS VERSIONS ARE COMPATIBLE WITH PCSPECIALIST LAPTOPS**

1. Format a flash drive to FAT32
2. Put the contents of this folder onto that USB
3. Copy the BIOS files you need to the USB
4. Boot to the USB
5. WAIT for the shell prompt to show (wait until after the 10s countdown)
6. Flash the BIOS by typing `F.nsh`. The laptop may reboot to a blue screen to flash the BIOS.
7. Wait for the laptop to reboot and complete the update
